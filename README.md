# 🏥 SGHSS – Sistema de Gestão Hospitalar

Sistema desenvolvido como projeto acadêmico para a disciplina de Projeto Multidisciplinar.

## 📌 Objetivo
O sistema tem como objetivo simular a gestão de um ambiente hospitalar, permitindo o cadastro de pacientes, agendamento de consultas, acesso a prontuários e realização de teleconsultas.

## ⚙️ Funcionalidades

- 🔐 Login e cadastro de usuários
- 👥 Cadastro e listagem de pacientes
- 📅 Agendamento de consultas
- 📋 Prontuário com evolução, prescrição e atestado
- 📹 Teleconsulta (simulação)
- 🔎 Filtro e busca de informações

## 🧠 Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- LocalStorage (simulação de banco de dados)

---

## 📌 Regras do sistema

- Os dados são armazenados no navegador utilizando LocalStorage
- Não há integração com banco de dados externo
- O sistema funciona apenas localmente
- Cada paciente é identificado pelo CPF
- As consultas são organizadas por médico, data e horário
